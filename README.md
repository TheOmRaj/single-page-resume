# HTML Resume Page Assignment

## Project Overview
This project is a single-page resume website built using pure HTML only, as part of the HTML Resume Page Assignment for the Web Development cohort.  
The goal was to recreate a provided resume layout using proper HTML structure and semantics, without using CSS.

---

## Features
- Single-page resume layout
- Clean and readable structure
- Uses only HTML5
- No CSS or external libraries
- Proper usage of:
  - Headings
  - Paragraphs
  - Lists
  - Tables
  - Horizontal rules

---

## Sections Included
- Header (Name & Title)
- About Me
- Skills
- Experience (Table format)
- Projects
- Education (Table format)
- Contact Details
- Footer

---

## Technologies Used
- HTML5 (No CSS or JavaScript used, as per assignment instructions)
